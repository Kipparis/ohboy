Витамин B
B1 нехватка вызывает: нервные срывы, психическую нестабильность
{
	орехи, гречка, кукурузка, фасоль, спаржа, картофель, отруби, печень
}
B2 нехватка вызывает: мышечную усталость, снижение жизненного тонуса
{
	молоко, мясо птицы, яйца, рыба
}
B3 нехватка вызывае: бессоницу, подавленное настроение, раздражительность
{
	мясо, яйца, орехи, семечки, зелёные овощи
}
B5 нехватка вызывает: плохое заживление ранок, частом затекании рук и ног
{
	кисломолочные продукты, зелёныце овощи, орехи, дрожжи, яичный желток
}
B6 нехватка вызывает: развитие сердчено сосудистых заболеваний
{
	говядина, яйца, молоко, зелёный перец, капуста
}
B9 необходим для нормального кроветворения, жирового обмена и укрепления иммунитета. Замедляет процесс старения организма.
{
	грибы, яичный желток, цветная капуста, лук, морковь, петрушка
}
B12 необходим для нормального функционирования нервной и имунной системы. Снижает концентрацию холестерина в крови, обеспечивает нормальное кроветворение, предупреждает малокровие
{
	морепродукты, мяцо птицы, яйца
}
